HTML

  • Anything that you want displayed on the site goes in the body tag
  • Title of the website goes in the head tag. This shows on the tab of the site

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The * applies the value to every element
  • The . applies value to class

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • commit- to push changes
  • default branch is main. create new branches to add your code and then merge
  • git allows version control- every version of the code is saved, you can go back to any version
  • git is local, github allows to connect and code with others, all team members can work on the same code without disrupting the others code
